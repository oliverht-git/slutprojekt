# Self-Improvement Hub

A unique and engaging web application designed to help users break bad habits, achieve goals, and improve themselves through gamification, challenges, and personalized features.

## 🎮 Unique Features

### Gamification System
- **XP & Levels**: Earn experience points for completing goals, maintaining habits, and finishing programs
- **Achievements & Badges**: Unlock badges for milestones like "First Goal", "Goal Master", "Habit Breaker"
- **Progress Visualization**: Level display in header with XP counter

### Challenge Mode
- **Pre-made Challenges**: Join intense challenges like "30 Days No Sugar", "Early Bird Challenge", "Gratitude Journal"
- **Personalized Difficulty**: Challenges with varying durations and commitments
- **Progress Tracking**: Visual progress bars for challenge completion

### AI-Powered Goal Generation
- **Smart Suggestions**: Generate personalized goals based on categories (Health, Productivity, Learning, Relationships, Finance)
- **Template-Based AI**: Intelligent goal suggestions tailored to your interests
- **One-Click Addition**: Easily add AI-generated goals to your list

### Daily Check-in & Reflection
- **Structured Reflection**: Daily journal with rating system (1-10) and progress questions
- **Challenge Documentation**: Note obstacles faced and lessons learned
- **History Tracking**: View past reflections to track growth over time

## 📊 Comprehensive Features

### User Accounts
- **Login/Register**: Create personal accounts to save your data securely in the browser
- **Personal Data Storage**: All your goals, habits, and progress are stored per user

### Goals & Positive Habits
- **Create Goals**: Add positive goals (e.g., "Work out 3 times per week") with completion tracking
- **Mark as Done**: Check off completed goals with XP rewards
- **Edit/Delete**: Modify or remove goals as needed
- **AI Generation**: Get personalized goal suggestions

### Bad Habits
- **Track Habits**: List habits you want to break
- **Streak Tracking**: Monitor how long you've been free from each habit
- **Daily Check-ins**: Mark progress each day to maintain streaks and earn XP

### Statistics Dashboard
- **Charts**: Visualize your progress with pie charts for goals and bar charts for habit streaks
- **Progress Bars**: See completion status for training programs and challenges

### Self-Improvement Strategies
- Browse a curated collection of proven strategies for personal growth

### Training Programs
- **Pre-made Programs**: Join ready-made challenges like 30-Day Exercise, Reading, and Meditation
- **Custom Programs**: Create your own training programs with personalized tasks
- **Progress Tracking**: Monitor daily completion with visual progress bars

### Notifications (Bonus)
- **Daily Reminders**: Get browser notifications to remind you about your goals (requires permission)
- **Achievement Alerts**: Instant notifications when you unlock badges or complete challenges

## 🎯 How to Use

1. **Start**: Register/login to create your account and begin earning XP
2. **Set Goals**: Add personal goals or use AI generation for suggestions
3. **Break Habits**: Track bad habits with streak counters and daily check-ins
4. **Join Challenges**: Take on intense challenges for accelerated growth
5. **Daily Reflection**: End each day with structured check-ins
6. **View Achievements**: Unlock badges and level up as you progress
7. **Track Stats**: Monitor your growth with detailed charts

## 🏆 Gamification Details

- **XP Rewards**:
  - Complete a goal: +20 XP
  - Daily habit check-in: +10 XP
  - Daily program/challenge progress: +5 XP
  - Join a challenge: +25 XP
  - Complete program/challenge: +50-100 XP bonus
  - Generate and add AI goal: +10 XP
  - Daily reflection: +5 XP

- **Achievements**:
  - First Goal: Create your first goal
  - Goal Master: Complete 5 goals
  - Habit Breaker: 3 habits with 7+ day streaks
  - Level Badges: Unlocked at each level milestone

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js for data visualization
- Local Storage API for data persistence
- Browser Notifications API

## 🔒 Privacy

All data is stored locally in your browser. No data is sent to external servers. User accounts are simulated using local storage for a personalized experience.
